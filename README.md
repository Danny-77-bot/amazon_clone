# Amazon Clone

## 📖 Project Description

This project is a front-end clone of the Amazon website built using HTML, CSS, and JavaScript. The goal of the project was to practice web development fundamentals, responsive layouts, DOM manipulation, and user interface design by recreating a familiar e-commerce platform.

## 🚀 Features

* Amazon-inspired homepage design
* Responsive navigation bar
* Product listing section
* Shopping cart interface
* Product cards with images, names, and prices
* Interactive buttons and user actions
* Responsive layout for different screen sizes

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
